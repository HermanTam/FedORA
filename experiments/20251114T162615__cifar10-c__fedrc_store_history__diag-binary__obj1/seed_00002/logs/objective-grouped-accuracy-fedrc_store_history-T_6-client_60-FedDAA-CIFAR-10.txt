LEGEND
--------------------------------------------------------------------------------
Personalization (P) Clients: P-clients on current local test (final slot).
Generalization (G) Clients: G-clients on previous (old) local test (final slot; falls back to current if missing).
Overall (All Clients): All clients on current local test (final slot).
Objective-aware OLD: Mix by objective, P→local current; G→old (final slot).
Objective-aware GLOBAL: Mix by objective, P→local current; G→fixed global test set (final slot).
GLOBAL TEST (Global P/G): P/G-clients on the fixed global test set (baseline).
P_on_local / G_on_local: P/G-clients on current local test (note: P_on_local equals P Clients).
P_on_global / G_on_global: P/G-clients on the fixed global test set (duplicates Global P/G).
Overall Global (All Clients): All clients on the fixed global test set (baseline).
ACROSS-TIME: For each slot t, average over clients; then summarize those T values (mean/std/min/median/max).
TASKS@FINAL (Rotation): Mean accuracy at final slot on rotation-stratified test sets (0°, 120°, 240°).
